

# Roadmap

## Design considerations
	
More details on my thoughts are available on the [development blog](https://averagebear.game.blog/):
- Dictatorial veto:
	1. Permadeath (No exceptions)
	2. No Phoenix/Casual mode
	3. No Ambush spawns
	4. NO DRAGONS
	
These features will not make it in my game, and it's final.

- Linear, chapter-based
	- Army movement on the world map only if it makes narrative sense
	- Certain story Events unlockable on world-map
	- Side stories
	- Enemy encounters on world Map are rare
- Items
	- Breakable
	- Becomes unusable "broken trash" item when broken
	- Weapon's remaining number of uses can be increased using whetstones
- Units have two hands
	- Swords & shields
	- Dual Wielding
	- Handedness/Ambidextry
	- Carry stuff in your hands to free up inventory space
	- Cavaliers need one hand on bridle
- Questionable features:
	- IronMan Mode

	Automatic game save + deletion of previous save after every move? After every chapter? Game Over deletes your savefile?

	- Forcible IronMan Mode

	Force player into IronMan mode

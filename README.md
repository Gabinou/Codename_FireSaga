Welcome to the Flame Ensign Maker (FEmaker) documentation.

This is intended to be a standalone, easy to use, strategy game maker, strongly inspired by Fire Emblem and to a lesser degree Advance wars. It will enable the construction of standalone games in executable format. I encourage any and all to use, modify, distribute this sofware or any piece of software produced, for commercial or non-commercial purposes. FEmaker should be free and open source.

I intend to make this software as compatible as can be with existing ROMHacking tools made by the Fire Emblem community, and common assets file formats. No assets of any kind come included with this software. We encourage any and all to make assets using free and open source tools:
